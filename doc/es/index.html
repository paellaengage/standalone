Paella Engage Player - Documentación de desarrollo

El reproductor Paella Engage es un reproductor de vídeo multistream capaz de reproducir múltiples streams de audio y vídeo simultáneamente, que además soporta la implementación de plugins. Está especialmente diseñado para grabaciones de clases, como píldoras Polimedia o clases de Matterhorn.

Índice

  • Paella Engage Player: una introducción a la arquitectura de Paella Engage Player.
  • Paella Plugins: arquitectura de plugins y recetas rápidas para crear tus propios plugin para Paella Engage Player.
  • Paella Extended: cómo utilizar Paella Engage como un portal web.
